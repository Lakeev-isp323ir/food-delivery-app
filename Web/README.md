# Веб-сайт (Web)

## Что будет находиться в этой папке
Исходный код веб-версии приложения для доставки еды.

## Планируемые компоненты/модули
- Главная страница с меню
- Страница оформления заказа
- Личный кабинет пользователя
- Административная панель (для ресторана)
- Корзина покупок

## Взаимодействие с другими частями проекта
Веб-сайт отправляет запросы к ServerApi, получает данные о меню,
отправляет заказы и управляет корзиной через API.

# Серверное API (ServerApi)

## Что будет находиться в этой папке
Серверная часть приложения, API для работы с базой данных.

## Планируемые компоненты/модули
- API для работы с меню (получение списка блюд)
- API для оформления заказов
- API для авторизации пользователей
- Модели базы данных
- Бизнес-логика обработки заказов

## Взаимодействие с другими частями проекта
ServerApi получает запросы от MobileApp и Web, обрабатывает их,
взаимодействует с базой данных и возвращает результат клиентам.

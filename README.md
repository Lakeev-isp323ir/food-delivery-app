# Food Delivery App (Приложение для доставки еды)

## Краткое описание идеи
Комплексное решение для доставки еды, включающее мобильное приложение для клиентов,
серверное API для обработки заказов и веб-сайт для просмотра меню.

## Структура репозитория
- **[MobileApp/](MobileApp/)** — мобильное приложение (iOS/Android)
- **[ServerApi/](ServerApi/)** — серверная часть и API
- **[Web/](Web/)** — веб-сайт

Подробное описание каждого компонента находится в соответствующих папках.

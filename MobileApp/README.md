# Мобильное приложение (MobileApp)

## Что будет находиться в этой папке
Исходный код мобильного приложения для iOS и Android.

## Планируемые компоненты/модули
- Экран входа и регистрации
- Главный экран со списком блюд
- Экран корзины
- Экран оформления заказа
- Профиль пользователя

## Взаимодействие с другими частями проекта
Мобильное приложение отправляет запросы к ServerApi для получения меню,
оформления заказов и авторизации пользователей.
